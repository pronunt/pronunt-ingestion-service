# Commit Conventions

Use Conventional Commits for this repository.

Format:

`type(scope): short summary`

Examples:

* `feat(api): add ingest endpoint`
* `fix(queue): handle GitHub rate limits`
* `refactor(service): simplify payload mapping`
* `docs(readme): update local setup steps`
* `chore(deps): bump FastAPI`
* `test(routes): add health endpoint tests`

Recommended types:

* `feat`
* `fix`
* `refactor`
* `docs`
* `chore`
* `test`
* `ci`
* `build`
